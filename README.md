# AI Personal Stylist（AI 私人形象顾问）

一个结合实时时尚趋势学习、身材分析、数字分身和虚拟试穿的 AI 穿搭智能体。

## 项目背景

很多人有自己喜欢的穿衣风格，但这些风格未必真正适合自己的身材比例、骨架和气质，导致：

- 收藏了大量穿搭图片，却难以复制效果
- 买了很多衣服，但上身效果不理想
- 不清楚问题出在版型、比例还是风格选择

## 项目目标

本项目希望构建一个 AI 私人形象顾问，帮助用户：

1. 学习最新时尚趋势
2. 分析个人身材与气质
3. 识别用户的审美偏好
4. 推荐更适合的穿搭方案
5. 生成数字分身
6. 展示虚拟试穿效果

## 核心创新

### 动态趋势知识库
持续从小红书、Pinterest 等平台提取最新流行趋势。

### 用户数字分身
根据照片与身体数据生成 3D Avatar，展示身材比例。

### 虚拟试穿
将推荐服装直接渲染到用户模型上，帮助用户在购买前预览效果。

## 系统流程

趋势学习 → 身材分析 → 风格识别 → 个性化推荐 → 数字分身 → 虚拟试穿

## 技术架构

- 大语言模型：OpenAI / Claude
- 图像分析：MediaPipe
- 3D 展示：Three.js
- 虚拟试穿：FASHN AI
- 前端：Next.js
- 后端：FastAPI

## 项目结构

- `knowledge/`：身材、风格与适配规则知识库
- `docs/`：PRD、系统设计与研究文档
- `examples/`：真实案例
- `assets/`：截图和演示素材
- `app/`：应用代码

## 项目愿景

让每个人都拥有一个真正理解自己身材、审美与生活场景的 AI 形象顾问。
